## Introduction

This project requires you to create an asynchronous web app that uses Web API and user data to dynamically update the UI for a Weather-Journal App.

## Usage

Install packages:

```sh
npm install
```

Start server:

```sh
node server.js
```

Open URL: http://localhost:8080
